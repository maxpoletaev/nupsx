nuPSX
PlayStation Emulator

nuPSX is an experimental PlayStation 1 emulator and debugger written in Zig. A passion project made for fun and learning, with no intention of becoming usable by the general public. If you are looking for an emulator to play games, check out DuckStation.

Try in Browser

This will run a WebAssembly build of nuPSX. Requires a legally obtained PlayStation BIOS image and a game executable or disc image. Files are only read locally in your browser and not uploaded anywhere. Not all games are expected to work, so check out the compatibility list.
